# Images Directory

Place your hackathon banner images, sponsor logos, and other graphics in this directory.

## Recommended Sizes

- **Banner Image**: 1200x400px (3:1 ratio)
- **Sponsor Logos**: 200x200px (square, transparent background recommended)

## Example Usage

```javascript
// In config.js
sponsors: [
    {
        name: "Sponsor Name",
        logo: "images/sponsor-logo.png",
        // ...
    }
]
```

Then reference in HTML:
```html
<img src="images/banner.jpg" alt="Hackathon Banner">
```
